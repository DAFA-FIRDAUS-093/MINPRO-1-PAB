# MINPRO-1-PAB


Aplikasi ini adalah aplikasi pencatatan data parkir yang dibuat menggunakan Flutter. Aplikasi ini memiliki fitur CRUD, yaitu menambahkan data kendaraan, menampilkan data, mengedit data, dan menghapus data. Data yang dicatat meliputi plat nomor, jenis kendaraan, dan jam masuk sehingga memudahkan pengelolaan kendaraan di area parkir.


Fitur - Fitur Aplikasi 
1. menambahkan data kendaraan
2. menampilkan data
3. mengedit data
4. menghapus data


Widget yang digunakan pada aplikasi ini antara lain MaterialApp, Scaffold, AppBar, Column, Padding, TextField, ElevatedButton, ListView.builder, Card, ListTile, Row, IconButton, dan AlertDialog.
